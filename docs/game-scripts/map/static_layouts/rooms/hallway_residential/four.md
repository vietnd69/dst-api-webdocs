# Four

> Based on game build **714014** | Last updated: 2026-03-03

## Overview
This file defines a static room layout for the *hallway_residential* family in the DST world generation system. It specifies tile placement (via a data grid) and structure for the "four" variant using the Tiled map format (v1.1, Lua encoding). It does not contain logic components, behaviors, or scripting — it is purely data describing a pre-built environment.

## Usage example
This file is not intended for direct runtime use by modders. It is loaded by the world generation system when placing a "hallway_residential/four" room. No manual instantiation or API calls are required.
