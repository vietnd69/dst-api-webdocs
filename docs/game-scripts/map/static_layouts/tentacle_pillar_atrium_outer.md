# Tentacle Pillar Atrium Outer

> Based on game build **714014** | Last updated: 2026-03-02

## Overview
This file defines a Tiled-based static layout used for the outer region of the tentacle pillar boss event in Don't Starve Together. It contains two layers: a background tile layer (`BG_TILES`) with tile ID data, and an object group (`FG_OBJECTS`) that specifies placements of decor items like fences, flowers, ferns, and stalagmites. This layout is consumed by the game’s world generation system to render the environment consistently during the event.

## Usage example
This component is not used directly by modders — it is loaded automatically by the game engine as part of the `tentacle_pillar_atrium` event layout chain. As such, it does not require programmatic usage.
